#linux command

- mv : 파일의 위치 변경
````
mv [이동할 파일명] [이동할 위치]
````
- cat : 파일안의 내용을 앞부터 표시
````
cat [내용 확인할 파일명]
````
- tail : 파일안의 내용을 뒤부터 표시
````
tail [내용 확인할 파일명]
````
###.bash_history
- 사용자가 실행했던 명령어 로그
````
vim ~/.bash_history
````
- 명령어 사용 -> 메모리 기록 -> 로그아웃 -> .bash_history -> 로그인 -> .bash_history -> 메모리
- 이전 사용 명령어 재실행
 - 화살표
 - !명령어 : 가장 최근에 실행한 명령어들 중 해당문자로 시작하는 명령어를 다시 한번 실행
 - !! : 가장 최근에 실행한 명령어를 다시 한번 실행
 - !번호 : 히스토리 목록중에 번호에 해당하는 명령어 실행
