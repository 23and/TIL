#코딩이란
- 주변에 있는 가전제품에서도 코딩이 들어간다
- 엘리베이터에서 해당 버튼을 누르면 이동을 하는 것도 코딩
- 자판기에 돈을 넣고 버튼을 눌렀을 때 음료수가 나오는 것에도 코딩이 들어간다
- 코딩은 컴퓨터한테 대화를 시도하는 하나의 언어, 도구, 수단
- 컴퓨터가 알아들을 수 있는 형태로 언어를 작성하는 것
- 생활 속 코딩
- 코딩은 삶을 좀 더 편안하게 해준다.

###알고리즘이란?
- 어떤 일에 우선순위를 둔다든지 그 다음에 어떤 일을 동시에 했을 때 무엇을 먼저 할 건지에 대한 조건들
- 탕수육을 조리할 때, 같은 탕수육이여도 만드는 방법이 다양한 것 처럼
- 방법을 순서대로 정해놔서 조리해내는 방법도 알고리즘

###코딩을 하면 좋은 것
- 문제에 어떻게 접근할 것인지
- 문제에서 무엇을 요구하는지, 어떻게 해결할 것인지
- 코딩을 통해 사고하게 된다면, 좀 더 합리적이고 좋은 방향으로 풀어나갈 수 있지 않을까
- 무에서 유를 창조하는 작업
- 조금 더 창의적인 생각을 한다면 사람들의 시간을 많이 절약하고, 좀 더 행복하게 만들어 줄 수 있다
