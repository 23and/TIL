#linux vi명령어
- 명령행 vi 파일명

###vi 3가지 모드
- 명령모드
 - ESC 키 vi 화면 좌측하단에 아무표시도 없음.
 - 화살표 방향키 커서 이동하기 
- 입력모드
 - i 키 
 - vi 화면 좌측하단에 -INSERT- 표시
- ex모드
 - Shift + : 키 vi 화면 좌측하단에 : 표시
 - ESC 키를 꼭 눌러서 명령모드로 바꾼후 해야 함

###명령어 종류
- u : undo
- x : 한 글자 삭제
- dd : 한 라인 삭제
- q! : 종료
- wq : 저장-> 종료
