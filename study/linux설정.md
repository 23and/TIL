#linux 환경
- 윈도우 OS 내에 설치하기 위해 VirtualBox 가상머신 설치
- 리눅스 OS 인 Ubuntu 다운
- 새로만들기 -> 종류 Linux -> 버전 Ubuntu 32bit
- 메모리 크기 설정 -> 지금 가상하드 드라이브 만들기 -> VDI -> 동적할당
- 동적할당 : 가상디스크를 사용할 때 고정된 최대크기까지 파일 크기가 커지지만, 사용량이 줄어들어도 자동적으로 작아지지 않는다.
- 고정크기 : 가상디스크를 만드는 데 더 오래걸리지만 사용할 때 더 빠르다.
- 설정 -> 저장소 -> 가상 CD/DVD 디스크파일 선택 -> 다운받은 Ubuntu ISO파일 추가
- 가상머신에 Ubuntu 설치
