#phpMyAdmin csv file 가져오기

- amazon에서 1년동안 무료로 사용할 수 있는 클라우드 서버를 제공한다.
- 프로젝트에서 임시적으로 amazon 서버를 사용하기로 했고, phpMyAdmin을 사용하고 있다.
- DB에 넣어야할 데이터 양이 많아서, phpMyAdmin에서 csv 파일로 저장한 데이터를 가져와서 한번에 데이터를 입력했다.

###방법
- 엑셀에서 컬럼별로 데이터를 입력한다.
- 저장할 때 확장자를 csv로 지정한다.
 - csv는 ,로 컬럼을 구분한다.
- 데이터베이스에서 테이블을 생성한다.
- 내보내기 메뉴
 - 가져올 파일에서 csv파일을 선택한다.
 - 형식에서 CSV using LOAD DATA를 선택한다.
 - '열(컬럼) 구분자'에 ','를 입력한다.
 - 실행
- AUTO_INCREMENT로 지정한 컬럼에는 임의로 다른 값을 넣어두었는데, 실행할 때 자동으로 값이 들어갔다.
